============
``progress``
============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Description
-----------

Returns the status of the synchronization process.

Request
-------

.. code-block:: http

   GET /api/v1/progress 

Response
--------

All response fields are wrapped in a top-level ``progress`` object.

Behavior
--------

- When the {+c2c-product-name+} is in the ``IDLE`` state, all output
  fields except ``state`` and ``canCommit`` are ``null``.
  
- When the {+c2c-product-name+} is in the ``PAUSED`` state, the
  ``lagTimeSeconds`` field is ``null``.
